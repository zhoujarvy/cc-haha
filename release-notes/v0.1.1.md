# Claude Code Haha v0.1.1
专项修复 Windows 桌面端启动异常与窗口控件问题。

## 修复内容
- 修复 Windows 自定义标题栏最小化/最大化/关闭按钮失效问题。
- 修复未安装 Git 环境下，桌面端无法启动聊天的问题。
- 优化 Windows 启动报错文案，精准展示失败原因，方便排查。

## 影响
Windows 端无需强制预装 Git 即可正常启动使用；依赖终端环境的特殊工具会给出友好提示，不再直接崩溃。

## 安装说明
**macOS**
首次打开如提示「已损坏」，执行：
```bash
xattr -cr /Applications/Claude\ Code\ Haha.app
```
